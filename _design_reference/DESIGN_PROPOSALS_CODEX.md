# Предложения по оформлению сайта «Кодислово» для Codex в VS Code

## Цель
Обновить светлую и тёмную темы сайта так, чтобы интерфейс выглядел современно, спокойно и цельно: сине-фиолетовая образовательная палитра, читаемый текст, мягкие карточки, единый стиль кнопок.

---

## 1. Общая идея оформления

Сайт должен сохранять текущую структуру:

- верхняя панель с логотипом;
- главный заголовок;
- две карточки: «Русский язык» и «Информатика»;
- кнопки классов;
- основные кнопки переходов;
- ссылка «Вход для учителя».

Изменить нужно в первую очередь цвета, контрастность, состояния кнопок и визуальную глубину карточек.

---

## 2. Светлая тема

### Палитра

| Назначение | Цвет |
|---|---|
| Фон страницы | `#F7F7FF` |
| Карточки | `#FFFFFF` |
| Основной текст | `#111827` |
| Вторичный текст | `#4B5563` |
| Основной акцент | `#4F46E5` |
| Акцент при наведении | `#4338CA` |
| Светлый акцентный фон | `#EEF2FF` |
| Рамки | `#DDD6FE` |
| Мягкая тень | `rgba(79,70,229,.12)` |

### Что изменить

1. Убрать красно-оранжевый текст с кнопок.
2. Основные кнопки сделать с белым текстом.
3. Вторичные кнопки сделать светло-фиолетовыми с тёмно-фиолетовым текстом.
4. Карточкам добавить тонкую фиолетовую рамку и мягкую тень.
5. Фон страницы сделать не чисто белым, а слегка фиолетовым.

---

## 3. Тёмная тема

### Палитра

| Назначение | Цвет |
|---|---|
| Фон страницы | `#0B1020` |
| Верхняя панель | `#080C18` |
| Карточки | `#121A33` |
| Вторичный блок | `#1A2447` |
| Основной текст | `#E9ECFF` |
| Вторичный текст | `#B8C0FF` |
| Основной акцент | `#4F46E5` |
| Акцент при наведении | `#6366F1` |
| Рамки | `#2B3A7A` |
| Мягкая тень | `rgba(0,0,0,.35)` |

### Что изменить

1. Не использовать полностью чёрный фон.
2. Убрать оранжевый текст с кнопок.
3. Основные кнопки сделать градиентными: `#4F46E5 → #6366F1`.
4. Вторичные кнопки сделать тёмно-синими, но светлее карточек.
5. Для текста использовать светло-лавандовый оттенок, а не чисто белый.
6. Добавить мягкое фиолетовое свечение в верхней части страницы.

---

## 4. Поведение элементов

### Карточки

При наведении:

- карточка слегка поднимается;
- рамка становится ярче;
- тень усиливается.

```css
.card:hover {
  transform: translateY(-2px);
  border-color: var(--primary-hover);
}
```

### Кнопки

Основные кнопки:

- фиолетовый градиент;
- белый текст;
- лёгкая тень;
- при наведении — небольшое поднятие.

Вторичные кнопки:

- мягкий фон;
- текст в цвет основного акцента;
- тонкая рамка.

Кнопки классов:

- компактные круглые «плашки»;
- без яркой заливки;
- при наведении — лёгкий фиолетовый фон.

---

## 5. Рекомендации для Codex

Нужно найти в проекте основной CSS-файл. Возможные варианты:

- `assets/css/ui.css`
- `assets/css/control-ui.css`
- `style.css`
- стили внутри `index.html`

После этого:

1. Вынести цвета в CSS-переменные.
2. Добавить отдельные переменные для светлой и тёмной темы.
3. Привязать тему к атрибуту `data-theme="dark"` на `html` или `body`.
4. Заменить прямые цвета в стилях на переменные.
5. Проверить читаемость кнопок, карточек и ссылок.

---

## 6. Главное правило

Не менять структуру HTML без необходимости. Основная задача — обновить оформление через CSS.
