# Задание для Codex в VS Code

Обнови цветовую гамму сайта «Кодислово».

## Что нужно сделать

1. Найди основной CSS-файл проекта.
2. Добавь CSS-переменные из файла `theme-tokens.css`.
3. Замени прямые цвета на переменные.
4. Исправь светлую и тёмную темы.
5. Убери оранжево-красный текст с кнопок.
6. Основные кнопки сделай фиолетовыми с белым текстом.
7. Вторичные кнопки сделай мягкими, с хорошей читаемостью.
8. Карточкам добавь мягкую рамку и тень.
9. В тёмной теме не используй абсолютно чёрный фон.
10. Сохрани текущую структуру сайта и ссылки.

## Визуальное направление

Стиль: современная образовательная платформа.

Цвета: синий + фиолетовый + мягкие светлые/тёмные оттенки.

Не использовать ярко-красный или оранжевый как основной цвет кнопок.

## Важное

Не ломай адаптивность. Проверь отображение на:

- компьютере;
- планшете;
- телефоне.

## Критерий готовности

Сайт должен выглядеть цельно в двух режимах:

- светлая тема;
- тёмная тема.

Текст должен хорошо читаться, а кнопки должны быть визуально понятными.
